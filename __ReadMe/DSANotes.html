Data Structures and Algorithms Notes

We summarize the performance characteristics of classic algorithms and data structures for sorting, priority queues, symbol tables, and graph processing.

We also summarize some of the mathematics useful in the analysis of algorithms, including commonly encountered functions; useful formulas and appoximations; properties of logarithms; asymptotic notations; and solutions to divide-and-conquer recurrences.

Properties of logarithms.

  • Definition:   logba=clogb⁡a=c means bc=abc=a. We refer to bb as the base of the logarithm.

  • Special cases:   logbb=1,logb1=0logb⁡b=1,logb⁡1=0

  • Inverse of exponential:   blogbx=xblogb⁡x=x

  • Product:   logb(x×y)=logbx+logbylogb⁡(x×y)=logb⁡x+logb⁡y

  • Division:   logb(x÷y)=logbx−logbylogb⁡(x÷y)=logb⁡x−logb⁡y

  • Finite product:   logb(x1×x2×…×xn)=logbx1+logbx2+…+logbxnlogb⁡(x1×x2×…×xn)=logb⁡x1+logb⁡x2+…+logb⁡xn

Master theorem.

Let a≥1a≥1, b≥2b≥2, and c>0c>0 and suppose that T(n)T(n) is a function on the non-negative integers that satisfies the divide-and-conquer recurrence
T(n)=aT(n/b)+Θ(nc)T(n)=aT(n/b)+Θ(nc)
with T(0)=0T(0)=0 and T(1)=Θ(1)T(1)=Θ(1), where n/bn/b means either ⌊n/b⌋⌊n/b⌋ or either ⌈n/b⌉⌈n/b⌉.
  • If c<logbac<logb⁡a, then T(n)=Θ(nlogba)T(n)=Θ(nlogb⁡a)
  • If c=logbac=logb⁡a, then T(n)=Θ(nclogn)T(n)=Θ(nclog⁡n)
  • If c>logbac>logb⁡a, then T(n)=Θ(nc)T(n)=Θ(nc)
Remark: there are many different versions of the master theorem. The Akra–Bazzi theorem is among the most powerful.
